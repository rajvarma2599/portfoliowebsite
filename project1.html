Mobile App Redesign for Fitness Tracker

Overview: Conducted user research and redesigned the UI/UX of a popular fitness tracking app, resulting in a 40% increase in user engagement. Created wireframes, high-fidelity prototypes in Figma, and conducted usability testing to ensure intuitive navigation and improved user satisfaction.

Key Features

  • User research and persona development
  • Wireframing and prototyping in Figma
  • Usability testing and iteration
  • Intuitive navigation design
  • Improved user engagement metrics

Tools Used

Figma, Adobe XD, UserTesting.com, Google Analytics

Results

40% increase in user engagement, improved user satisfaction scores, and positive feedback on app store reviews.
